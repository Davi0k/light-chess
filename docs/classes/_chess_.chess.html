import

  • import(fen: string): void
    • Defined in chess.ts:188

    Imports a FEN expression format replacing the current match.

    throws

    If the FEN expression is not valid and does not match with the RegExp, the method will throw an Error.

    remarks

    The RegExp pattern used for FEN validation is: \s*^(((?:[rnbqkpRNBQKP1-8]+\/){7})[rnbqkpRNBQKP1-8]+)\s([b|w])\s([K|Q|k|q|-]{1,4})\s(-|[a-h][1-8])\s(\d+\s\d+)$

    Parameters

    • fen: string

      The FEN expression which will be imported (example: e2).

    Returns void
